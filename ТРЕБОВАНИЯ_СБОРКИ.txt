✅ РЕЗУЛЬТАТ ДОЛЖЕН БЫТЬ:
  1. Команда: npm run build
  2. Vite собирает src/**/*.ts с @core/@domain/@infrastructure алиасами
  3. Vite ПРЕОБРАЗУЕТ алиасы в относительные пути в dist/
  4. Результат: dist/**/*.js с import ... from '../core/index.js' (не @core)
  5. Приложение запускается: node app.js diagnostics

✅ ПРОВЕРКА:
  - head -20 dist/view/cli/commands/diagnostics.js должен содержать
    relative paths вроде './core/index.js' или '../../../core/index.js'
  - ❌ НЕ должно быть '@core' в скомпилированных файлах
